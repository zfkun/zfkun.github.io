解决VMware Tools installation cannot be started manually while Easy Install is in progress.

2011-04-19
1分钟阅读时长
193字
阅读
很多尝试VMware 7虚拟机的朋友都会发现，用Ghost xp安装Windows以后，在开始安装VMware Tools时，会弹出显示如下内容的警告框：
VMware Tools installation cannot be started manually while Easy Install is in progress.
解决方法：
在开启虚拟机操作系统的情况下，在虚拟机设置中更改floppy为auto detect。然后点击菜单中的安装VMWare Tools，虚拟操作系统中的光驱里就会有VMWare Tools的安装程序了，自行安装即
