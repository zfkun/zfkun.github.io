主要记录在 Blender 工程中较为常见的疑难问题解决方法及一些实用操作方式

玩blender离不开快捷键的熟悉程度

常见问题

法线朝向不对引起渲染效果错误

要操作法线, 需先进入 编辑模式 (tab) 鼠标点击 网格 选项卡打开菜单, 或使用以下捷键:

  • 法向 (菜单): tab > alt + N
  • 重新计算外侧 (命令): tab > shift + N
  • 重新计算内侧 (命令): tab > shift + command + N

shift + c 后, 滚轮缩放场景失效BUG解决

按 ~, 选择 查看所选

活动相机对齐到当前对象

物体模式 下选中物体, 按 shift + commnad + 0

点、线、面快速切换

若开启了 输入 - 键盘 - 模拟数字键盘 选项, 则会因快捷键冲突, 造成 1, 2, 3 快速切换点、线、面指令失效（编辑模式 下）, 变成了 视图切换

选择 + 移动

类似 c4d 等软件, 选择物体的同时, 可以显示 移动坐标轴 允许移动操作 (G)

点击 Gizmos 下拉菜单, 物体Gizmo 下勾选 移动 即可

非选中物体消失不可见

因 处于 局部视图模式 (/键), 当前只居中放大显示选中物体, 类似于 c4d 的 独显模式 (solo)

此时, 全局显示(Home键) 操作也会失效, 再按 /键 退出 局部视图模式后可正常

物体原点修改

右键 - 设置原点 (物体模式下)

按 ctrl + alt + X (3D Viewport Pie Menus 插件开启下)

游标(选中项)位置对齐

按 shift + S

创建边面

类似 C4D 的 封闭多边形孔洞 的效果

多选点或边或面，按 F 即可完成自动填充 (编辑模式下)

圆环拓扑

点、线、面 快速按圆环拓扑 (掏洞拓扑常用)

借助 球形化 指令, shift + alt + S

借助 LoopTools 插件, 右键 > 圆环

常用操作

显示全部

按 Home

居中显示当前选择

按 ~, 选择 查看所选

游标(选中项)位置对齐

按 shift + S

游标归位世界中心

按 shift + C

滑动

点、边、面 均适用

按 shift + V (编辑模式下)

环切 + 滑动

边、面 适用

按 command + R (编辑模式下)

删除或融并

按 X

选择方式

框选、刷选、套索 等

按 W 依次切换

按 shift + 空格 > B 快速切换到 框选 按 shift + 空格 > C 快速切换到 刷选 按 shift + 空格 > L 快速切换到 套索

常用插件

万物有灵

神器 https://shimo.im/docs/qjXWwYkGKccgXVgd/read

M3

提效工具集 https://github.com/machin3io/MACHIN3tools

BoxCutter

非破坏性布尔建模神器

Exoside QuadRemesher

重拓扑神器, 可以将原来三角面的模型，重新转换成四边形模型

网上大都是 WIN 版本的, 而 MAC 版我就用 C4D 正在使用的版本里的引擎, 也是通用的 (毕竟 Quad Remesher 引擎是独立运行的)

链接: https://pan.baidu.com/s/14RjEvr4x_hBqrimEjFTjIw 提取码: 7dp4

安装方法

  1. 将 quad_remesher_1_2 目录复制到 ~/Library/Application\ Support/Blender/3.0/scripts/addons 目录下 (相当于手工注册插件)
  2. 回到 Blender 打开 偏好设置 - 插件, 搜索 Quad 找到 Mesh: Quad Remesher Bridge 打钩 (启用插件)
  3. 场景中随便选中一个物体, 按 N 打开导航, 切换到 Quad Remesh 面板, 点击 REMESH IT 按钮 (触发引擎初始化下载及安装)
  4. 将 QuadRemesherEngine 目录内所有文件复制到 /Users/Shared/Exoside/QuadRemesher/Datas_Blender/QuadRemesherEngine_1.0 下覆盖即可
